## 🔧 Usage

### Quick Start
```bash
# Install plugin
claude install plugin https://github.com/khiwniti/claude-plugin.git

# Or use locally
git clone https://github.com/khiwniti/claude-plugin.git
cd claude-plugin/ai-agent-saas-expert
cc --plugin-dir .
```

### Common Workflows

**Architecture Review:**
```
User: "Review my monorepo architecture"
→ architecture-auditor agent activates
→ Analyzes structure, provides health score
```

**Production Readiness:**
```
User: "Is my project ready for production?"
→ production-readiness agent activates
→ Comprehensive audit with prioritized issues
```

**Security Audit:**
